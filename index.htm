What is a sosreport?

 

A sosreport is a command in Linux which collects system configuration and diagnostic information of your Linux VM. For example: running kernel version, loaded modules, and system and service configuration files.

 

This was originally only for Red Hat, but is now generally available in most Linux distributions

 

 

 

How to use it?

 

To run sosreport, the sos package should be installed if using Red Hat and sosreport package for Ubuntu. If for any reason the required package is not installed, then use below commands to install it:

 

Redhat:

 

# yum install sos

 

Ubuntu:

 

# apt-get install sosreport
